📄 Application Form (Dynamic)

A responsive and dynamic application form built using HTML, CSS, and JavaScript. This project demonstrates form validation, dynamic field handling, and user-friendly UI design.

🚀 Features
🧾 User-friendly application form UI
✅ Client-side form validation
⚡ Dynamic input fields (add/remove sections)
📱 Fully responsive design
🔔 Real-time error handling and feedback
💡 Clean and modern layout
🛠️ Technologies Used

HTML5 – Structure of the form
CSS3 – Styling and responsiveness
JavaScript (Vanilla JS) – Dynamic behavior and validation
